# BIST50 Correlation Analysis
This project analyzes the correlation structure of BIST50 stocks that remained in the index for the past 2 years.

## Tools Used
- Python
- Pandas
- NumPy
- yFinance
- Matplotlib
- Seaborn

## Main Insight
The analysis shows that banking stocks have very high positive correlations.

This means adding many banking stocks into the same portfolio may increase concentration risk instead of improving diversification.
